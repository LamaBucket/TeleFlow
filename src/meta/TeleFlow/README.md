# TeleFlow

**TeleFlow** is a Telegram bot framework for .NET for **structured conversations**: update pipeline + commands + chat sessions + multi-step flows.

📚 Docs: https://lamabucket.github.io/TeleFlow/  
💻 GitHub: https://github.com/LamaBucket/TeleFlow  

---

## Highlights

- Update pipeline (middleware-style)
- Stateless commands + stateful multi-step flows
- `ChatSession` to resume dialogs across updates
- `CommandResult` + interpreters for navigation / step transitions
- Built-in steps: `TextInput`, `ContactInput`, `DateInput`, `ListSelection`
